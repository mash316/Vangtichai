Project Name: VangtiChai

Tested Devices
--------------
The application has been tested on the following devices and screen configurations to ensure full responsiveness and compliance with the Greedy Change-Making algorithm requirements:

Required Devices:
- Pixel XL (411 x 731 dp): Tested in both Portrait and Landscape orientations.
- Nexus 10 (800 x 1280 dp): Tested in both Portrait and Landscape orientations.

Additional Devices:
- Pixel 8: Tested for modern smartphone aspect ratios.
- Pixel 9: Tested for high-density displays.
- Pixel_XL_assignment (AVD): Custom virtual device used for primary development and layout verification.
- sdk gphone16k x86 64 (emulator-5554): Used for general Android API 37 functionality testing.

The layouts have been verified to adapt gracefully between phone and tablet form factors, reorganizing the keypad and denomination tables to optimize screen real estate in both orientations.
